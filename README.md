# devops_practice
This repository contains my solutions and work for various DevOps projects, ranging from beginner to advanced levels. These projects are based on the roadmap.sh DevOps project ideas.

# Projects List
## Scripting Bash
- [Server Performance Stats Script](https://roadmap.sh/projects/server-stats): A bash script that analyzes and displays real-time server performance statistics including CPU, memory, disk usage, and top running processes. folder: [scripting/Server-Performance-Stats-Script](scripting/Server-Performance-Stats-Script/README.md)
- [Nginx Log Analyser](https://roadmap.sh/projects/nginx-log-analyser): A simple command-line tool to analyze Nginx access logs, providing insights such as top IP addresses, requested paths, response status codes, and user agents. folder: [scripting/Nginx-Log-Analyser](scripting/Nginx-Log-Analyser/Readme.md)
